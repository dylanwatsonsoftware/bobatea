Download

Add Boba Tea to your project using Gradle:

repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.dylanwatsonsoftware:bobatea:0.1'
}

Examples

Explore what you can build with Boba Tea's component-based architecture.

Loading Indicator

runLoading("Loading yo!") {
    delay(3000)
}

Multi-select List

val multiSelections = selectMultipleFromList(
    question = "What are all your favourite numbers?",
    options = listOf("one", "two", "three"),
    borderStyle = BorderStyle.SINGLE,
    padding = 1
)

Single Select List

val selection = selectFromList(
    question = "What's your favourite number?",
    options = listOf("one", "two", "three"),
    borderStyle = BorderStyle.ROUNDED,
    color = YELLOW
)

Co-ordinate Selection

Easily create custom navigation and interactive grids.

Click to Expand

Interactive components that expand when clicked or when the space/enter key is pressed.

expandable(
    title = "Click me to see more!",
    content = "You expanded the section! Surprise!",
    borderStyle = BorderStyle.SINGLE,
    padding = 1
)

Box Model Layout

Flexible box model to format your text in the terminal with borders, padding, and margins.

val box = Box(
    content = "Welcome to Boba Tea!",
    padding = 1,
    borderStyle = BorderStyle.ROUNDED,
    color = CYAN
)
println(box.render())

Advanced Component Usage

All UI components inherit from BobaComponent, providing a consistent API for interaction and rendering.

val myComp = SelectionList(
    question = "Styled List",
    options = listOf("Option 1", "Option 2"),
    padding = 1,
    margin = 2,
    borderStyle = BorderStyle.DOUBLE,
    color = BLUE
)
val result = myComp.interact()
